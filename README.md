## Color Palettes

I use a subset of colors from [vimcolorscheme](https://vimcolorschemes.com/).

1. [Gruvbox](https://github.com/morhetz/gruvbox)

```js

	'#282828', '#ebdbb2', '#fb4934', '#b8bb26', '#fabd2f',
	'#83a598', '#d3869b', '#8ec07c', '#a89984', '#fe8019',
];
```
